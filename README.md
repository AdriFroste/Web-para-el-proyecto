# ♖ Ascend Tower

> *¿Lograras esquivar las trampas?*

**Ascend Tower** es un plataformas de precisión 2D donde el protagonista, movido por la ambición de dejar atrás la pobreza, se adentra en una legendaria torre que custodia incalculables tesoros en cada una de sus plantas. Cada piso es un reto nuevo — trampas, sierras, pinchos y saltos imposibles se interponen entre tú y la fortuna soñada.

## 🎮 ¿De qué va?

El juego se basa en un control refinado y una dificultad progresiva. El objetivo de cada nivel es sencillo: llega al final evitando todas las trampas. Fácil de entender, difícil de dominar.

- 💀 **Muerte instantánea** — un pincho, una sierra, un mal salto y empiezas de nuevo
- 📈 **Dificultad progresiva** — cada piso es más exigente que el anterior
- 🔥 **Modo Extremo** — para los que completen el juego y quieran sufrir un poco más
- 🏃 **Speedrun-friendly** — controles precisos pensados para jugadores que quieren mejorar su tiempo

## 🕹️ Controles

| Acción | Tecla |
|--------|-------|
| Mover | `A` / `D` o flechas |
| Saltar | `Espacio` |
| Wall jump | Saltar hacia la pared |

> Mantén el salto para saltar más alto. Suéltalo antes para un salto corto.

## 🎨 Estética y sonido

El juego tiene una **estética retro arcade** con pixel art y un **shader VHS/CRT** que añade escáner, estática y efecto ojo de pez para una inmersión total en el rollo clásico. Cada piso tiene su propia banda sonora generada con **Suno AI**.
